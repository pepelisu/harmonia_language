## Grammar

| Element | Controls | Options |
|---------|----------|---------|
| **Octave** | Tense | 2=ancient, 3=past, 4=present, 5=future, 6=eternal |
| **Dynamic** | Register | pp=secret, mp=normal, f=urgent, ff=command |
| **Meter** | Speech mode | 4/4=testimony, 3/4=spell, 6/8=story, 2/4=order |
| **Inversion** | Case | Root=subject, 1st=object, 2nd=modifier |
| **Instrument** | Speaker | Piano=human, strings=community, celesta=otherworld |
| **Silence** | Punctuation | Short=comma, long=period, very long=negation |

## Quick Translation Process

1. **Key** = What is this about? (root) + What stance? (major/minor)
2. **Meter** = What kind of speech? (testimony/spell/story/order)
3. **Map concepts to roots** — one root per key idea
4. **Choose qualities** — affirmed/denied/questioned/wondrous/dangerous
5. **Set dynamics** — whispered? declared? commanded?
6. **Add connectors** — bass walks + melodic bridges + common tones
7. **Play it** — does it flow? adjust until it does
